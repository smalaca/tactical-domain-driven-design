# Tactical Domain-Driven Design

## Goal of the repository

The goal of this repository is to demonstrate the usage of Domain-Driven Design (DDD) tactical patterns combined with the Ports and Adapters architecture. It aims to provide a clear implementation where a bounded context is fully realized, showing how to maintain a clean separation between the domain logic and technical concerns.
